日期和委員會是雙重 filter，像 google+ 的 circles 管理那樣，hover 任一項目，另一區有交集的項目會被 highlight 起來

第一個被使用者選取的區塊，會變成 primary filter，另一區則變成 secondary filter。primary filter 的 selected item 有橘色邊框。可多選

secondary filter 裡面，跟 primary filter 選取項目有交集的按鈕會變成 active，其他則變成 disabled

secondary filter 裡面 active 的按鈕切 on/off，底下的議案就會選擇性顯示

primary filter 的 selected item 再按一下可取消選取，全部取消選取後，才能重新選擇 primary filter

filter 預設值：日期：今天，委員會：全部
